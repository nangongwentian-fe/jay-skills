# 任务后经验维护规则（post-task-learning-review）

完成非平凡任务后，主动使用 $post-task-learning-review 判断本次经验是否应在项目文档、memory、已有 skill 或新 skill 中新增、更新、合并或删除。

项目内经验优先写入项目文档，Codex memory 只作为跨会话补充；写项目文档时遵守渐进式披露。

默认只提出建议；写入 memory、创建/修改 skill、修改项目文档时遵守对应权限和用户授权。
