# 详细工作流程

## 完整触发场景

### 显式触发
- 用户说"记住xxx"、"记录下来"
- 用户问"之前怎样"、"上次那个"
- 完成重要任务
- 会话结束有新内容

### 自动触发
检测到以下信息时，主动提示用户：
- API key、token（不管是否打码）
- 账号、密码
- 项目背景、技术决策
- 用户偏好、习惯
- 重要联系人

## 敏感信息处理

### 绝对不记录
- 真实密码
- 银行卡号
- API 真实密钥

### 可以记录
- 账号名/ID
- 配置项名称
- 技术方案

### 自动检测
```bash
grep -E "(sk-|ghp_|eyJh|bearer|password|secret)" ~/.persistent-memory/
```

检测到时提示：
```
⚠️ 检测到敏感信息，是否需要记录？（建议只记录账号名）
```

## 整理记忆

### 定期整理
- 每次心跳或每周回顾近期日记
- 将重要内容合并到 MEMORY.md
- 删除过时信息

### 整理检查清单
1. 最近 7 天日记有哪些新知识？
2. 有没有待办事项完成？
3. 技术知识有没有过时？
4. 账号信息有没有变更？
